# Painel Carvalho (Mock Front v4)

Mock visual em **HTML + CSS + JS** (sem backend) para apresentar o futuro painel online.

## Como rodar (Windows)
1. Extraia a pasta
2. Abra um terminal dentro da pasta (onde está `index.html`)
3. Rode:
   ```bat
   python -m http.server 8000
   ```
4. Acesse:
   - http://localhost:8000

> Se `python` não existir, tente `py -m http.server 8000`.

## Login (mock)
- e-mail: qualquer
- senha: **carvalho**
